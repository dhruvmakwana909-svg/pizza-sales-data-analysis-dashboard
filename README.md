# Pizza Sales Data Analysis Dashboard

## Project Overview
This project analyzes pizza sales data and presents insights through an interactive Excel dashboard.

## Key Metrics
- Total Revenue: $69,793
- Total Orders: 1845
- Total Pizza Sold: 4232
- Average Order Value: $37.83
- Average Pizza per Order: 2.29

## Dashboard Insights
- Daily Order Trends
- Hourly Sales Trends
- Sales by Pizza Category
- Sales by Pizza Size
- Top 5 Best Selling Pizzas
- Top 5 Worst Selling Pizzas

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Data Cleaning
- Data Visualization

## Skills Demonstrated
Data Analysis, Business Intelligence, Dashboard Development, Data Visualization
